# Project Overview

## 1. Project Name

Personal_Web。

## 2. Project Purpose

Personal_Web 是一个长期维护的个人网站，不是一次性静态页面。

它同时包含两种用途：

- 公开展示：面向普通访客展示个人介绍、作品、文章、项目、联系方式等公开内容；
- 私人工具：面向管理员和授权用户，提供长期扩展的个人工具中心。

当前阶段以确认设计方向、边界和文档体系为主，不急于写业务代码。

## 3. Two Main Areas

- Public Visitor Site
- Private Personal Hub

Public Visitor Site 是访客默认看到的公开网站。

Private Personal Hub 是登录后使用的个人工具中心。

这两个区域可以共享整体视觉语言，但访问边界、权限边界、数据边界和代码边界必须清晰。

## 4. Current Confirmed Direction

- 域名首页默认进入公开访客网站。
- 公开主页上存在一个隐藏图案入口。
- 点击隐藏图案进入登录页。
- 登录后进入个人工具中心。
- 管理员可以创建账号。
- 管理员可以给不同用户分配权限。
- 个人工具中心会长期扩展多个子应用。
- 设计文档会持续调整和完善。
- 代码必须跟随文档，而不是随意发挥。
