# Public Site Design

## 1. Purpose

公开访客网站是所有访客默认看到的页面。

它用于展示个人形象、项目、作品、文章或其他公开内容。具体展示内容当前尚未完全确定，未确定部分必须标记为 TBD。

## 2. Confirmed Requirements

- 打开域名后默认进入公开访客网站。
- 公开网站不应该像后台系统。
- 公开网站应该像正常个人主页。
- 页面中有一个隐藏图案，这个图案是进入登录页的入口。
- 隐藏入口不能明显写成“登录”按钮。
- 隐藏入口应该更像彩蛋或个人风格元素。

## 3. Hidden Entrance Pattern

隐藏入口的设计方向可以是：

- 页面角落的小图案
- Logo 的某个区域
- 钥匙孔
- 星星
- 月亮
- 像素门
- 猫爪
- 气泡
- 其他个人化图案

目前具体图案 TBD。

隐藏入口只是从公开主页进入登录页的视觉入口，不是安全措施。

当前静态阶段，隐藏入口链接到 `login.html` 登录占位页。该占位页不提供真实登录、认证或权限能力。
