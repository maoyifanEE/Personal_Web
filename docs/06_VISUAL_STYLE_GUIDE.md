# Visual Style Guide

## 1. Overall Feeling

当前方向：

干净、温和、有个人气质、有一点隐藏感和私人空间感。

## 2. Public Site Style

公开网站应该更像个人主页，不应该像管理后台。

建议方向：

- 简洁
- 有留白
- 有轻微动效
- 有个人符号
- 隐藏入口自然融入页面

## 3. Personal Hub Style

个人工具中心应该更偏工具感，但不要像传统企业后台。

建议方向：

- 卡片式布局
- 清晰导航
- 统一按钮样式
- 统一字体
- 统一间距
- 电脑和手机都好用
